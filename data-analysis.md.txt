# 데이터 분석 프로세스
## 1. 문제 정의
- 목적 : 최종적으로 얻고 싶은 것
- 목표 : 목적을 구체화한 것, 수단
## 2. 데이터 수집
- 데이터 정의
- 데이터 수집

### 데이터 - 범주형(명목형, 순위형), 수치형(이산형, 연속형)
수집을 위해 실험계획법을 활용하면 시간, 돈, 인력 절약 가능

## 3. 데이터 분석
### 데이터 전처리
- 결측치 : 데이터 삭제, 평균/중앙값/최빈값 등 대푯값으로 채우기
- 이상치 : 데이터 삭제, 평활화(smoothing)을 해 중간값으로 대체
- 중복값 : 하나만 남기고 삭제
### 기초 통계 분석
- 대푯값과 상관 계수를 계산하여 데이터 특성 파악
- 의미 없는 데이터 제거 및 데이터 분석에 필요한 변수 선택
### 모델 구축 및 평가 과정
- 모델 : 각 아록리즘이 데이터 분석을 진행하면서 생성하는 로직이나 수식
- 통계적 알고리즘 : 분산분석, 회귀분석, 주성분분석, 요인분석, 판별분석
- 인공지능 알고리즘 : 머신러닝이나 데이터 마이닝 분야
- test와 train을 나눠서 성능 평가

## 4. 검증 및 고찰
- 분석 결과 : 분석 보고서, 회귀식과 같은 모델
